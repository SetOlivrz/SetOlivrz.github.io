Dreamscape

PROJECT DESCRIPTION

A 3D survival game where you play as a kid who tries to escape her labyrinth infested nightmare. Made under the Game Jam theme of "Lost and Found".

This project was our entry for the Game Jam with the theme of "Lost and Found".

PROJECT SPECIFICATIONS

TYPE: Game
GENRE: Survival, Horror
PLATFORM: Microsoft Windows
ENGINE: Unity Engine
