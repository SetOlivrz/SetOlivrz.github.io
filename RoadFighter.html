Road Fighter Remake

PROJECT DESCRIPTION

A remake of "Road Fighter", a 2D car-based arcade game developed by Konami in 1984.

This project applies the practice of object-oriented programming and was developed using various features SMFL (Simple and Fast Multimedia Library).

PROJECT SPECIFICATIONS

TYPE: Game
GENRE: Arcade
PLATFORM: Microsoft Windows
IDE: Visual Studio - C++ | SFML
YEAR PUBLISHED: 2021
