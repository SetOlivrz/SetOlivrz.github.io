Square One

PROJECT DESCRIPTION

A rail-shooter mobile game where you play as a quirky inventor who makes unconventional devices in order to fight off the 2D creatures who infiltrated ours. "

This project was one of our major outputs in GDAPDEV (Web and Mobile Game Development) course where we are tasked to create a rail-shooter type of game that integrates the different features of a mobile platform
